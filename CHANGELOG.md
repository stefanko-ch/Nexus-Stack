## [v0.9.0](https://github.com/stefanko-ch/Nexus-Stack/releases/tag/v0.9.0) (2026-01-13)

### ‚ú® Features
- **Migrate state storage from local to Cloudflare R2** (#14)
  - Automatic R2 bucket creation via `scripts/init-r2-state.sh`
  - S3-compatible backend with state locking (`use_lockfile`)
  - Split secrets (`.env`) from settings (`config.tfvars`) for CI/CD support
  - Add `.env.example` template with required `TF_VAR_*` variables
  - Update Makefile to auto-load `.env` and support `make destroy`
  - `make destroy` now deletes state files before R2 bucket cleanup
  - Add Workers R2 Storage:Edit to required Cloudflare API permissions

- Document `.env` format limitations in Makefile
- Fix AWS credentials mapping in destroy target (`R2_*` ‚Üí `AWS_*`)
